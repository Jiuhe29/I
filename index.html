开发理念

指导我工作的核心原则

代码清晰

遵循 SOLID 原则，编写具备可读性和可维护性的代码。统一命名风格，模块职责分明。

性能优先

从 SQL 查询优化到静态资源懒加载，始终关注系统响应速度与资源利用效率。

合理分层

划分实体、DTO、服务、控制器层，保持职责清晰、代码高内聚低耦合。

极致体验

注重视觉统一与交互细节，运用 Tailwind CSS 构建丝滑、简约的 UI 界面。

数据为本

重视数据结构设计与约束，运用 EF Core 数据库优先策略构建可靠的数据基础。

工具高效

善用 VS、VS Code、Swagger、Postman 等工具提升开发效率，优化调试与测试流程。
